# ProcrastinAide
The application that combats procrastination.

ProcrastinAide is a student-focused productivity app designed to help users overcome procrastination by turning task management into a structured, motivating experience. The app combines proven productivity techniques, like focused work sessions and visual progress tracking, with a clean, student-friendly interface to help users stay accountable and build better study habits.

# Installation + Setup Guide
Developed using Ionic, Angular + Node Js Version v22.20.0, and FireBase (backend)
To run this application, you will need to install the angular package.

1. Clone the repository.
2. You will need to install Node JS first to use Angular and run the 'ng' commands.
3. Here is the installation guide for Node Js: https://nodejs.org/en/download
4. Run 'npm install -g @angular/cli' (minus quotes) in your terminal.
5. Also run 'npm install chart.js' (minus quotes) to avoid build failure.
6. VS Code is a good code editor to run this command on.
7. Here is the installation guide for VS Code: https://code.visualstudio.com/
8. Once your dependencies are installed run 'ng serve' (minus quotes) on your terminal to launch the application.
9. Click the local host link and now you can see the application!
